배열은 연관된 정보를 그룹화하는데 사용한다.

- 동일한 자료형만 묶어서 저장가능
	- int [] a; int 자료형만 저장가능
- 생성시 크기를 지정 (이후 크기 변경불가)

배열의 객체 생성
	여러 개의 값을 저장할 수 있는 공간
		
		ex)
		자료형[] a = new 자료형[배열의 길이];


배열의 생성 및  값 대입
	ex)
	
		 int[] a = new int[3];
			a[0] = 3;
			a[1] = 4;
			a[2] = 5;
		int[] a = new int[] {3,4,5};
		int[] a = {3,4,5}; // 이방법은 비추천


>[!important]
>[[힙영역 (heap)]]에 저장됨
