
==클래스 영역에서 타입 앞에 static이 붙는 변수 객체를 공유하는 변수로 여러 객체에서 공통으로 사용하고 싶을 때 정의==

>[!warning]  
>접근 방법: 클래스명.클래스변수명
>	ex) class Exem에 static 변수로 int 형태의 age가 있으면 
>		Exem.age = 1; 이런 식으로 접근이 가능
>		인스턴스 변수의 경우 객체(=인스턴스)를 생성하고 참조 변수를 통해서만 접근이 가능한데 ==클래스 변수는 객체를 생성하지 않아도 클래스명으로 바로 접근이 가능함==


>[!important]
>[[메서드영역 (method)]]에 저장
