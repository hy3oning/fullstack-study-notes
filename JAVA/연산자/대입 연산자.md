

|형태|의미|
|---|---|
|`=`|오른쪽 값을 왼쪽 변수에 대입|

|연산자|의미|풀어쓴 형태|
|---|---|---|
|`+=`|더해서 대입|`a = a + b`|
|`-=`|빼서 대입|`a = a - b`|
|`*=`|곱해서 대입|`a = a * b`|
|`/=`|나눠서 대입|`a = a / b`|
|`%=`|나머지 대입|`a = a % b`|

|연산자|의미|
|---|---|
|`&=`|비트 AND 후 대입|
|`|=`|
|`^=`|비트 XOR 후 대입|
|`<<=`|왼쪽 시프트 후 대입|
|`>>=`|오른쪽 시프트 후 대입|
|`>>>=`|부호 없는 오른쪽 시프트 후 대입|
