
==부모클래스(상위 클래스)에 정의된 메서드를
자식클래스(하위 클래스)에서 재정의하는 것. 덮어쓰기==

>[!important]
>⭐오버라이딩 조건
>- 상위 클래스와 하위 클래스 관계에서만 성립할 수 있다.
>- 메서드의 선언부(메서드 이름, 매개변수, 반환 타입)가 상위 클래스의 메서드랑 일치해야 한다.
>- 접근 제어자의 범위가 상위 클래스의 메서드보다 같거나 넓어야 한다.
>- 예외는 상위 클래스의 메서드보다 많이 선언할 수 없다.
>- 부모클래스의 메서드와 시그너처 및 리턴 타입 동일

>[!warning]
>인스턴스 필드는 오버라이딩이 되지 않음
>static 필드는 오버라이딩이 되지 않음
