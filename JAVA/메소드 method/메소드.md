메소드는 클래스의 내부에 존재하는 영역으로, 특정 기능을 하는 코드를 묶어서 나타내는 것이다. 즉, 특정 기능을 수행하기 위한 코드들의 집합체이다.


반환타입
- 메서드는 처리 결과값을 반환한다. 처리 결과는 값의 형태로 된 데이터를 뜻하며, 이 데이터가 반환 값이다.
- 값은 여러 종류로 나뉜 데이터로 정수, 문자, boolean,등 다양하다.
- void의 경우 없음을 뜻하여 반환 값이 없다는 것을 의미한다.


메인 메서드

	public static void main(String[] args){
	}
- void : 반환 타입으로 메인 메서드는 반환을 하지 않는다.
- main : 메서드의 이름이 main이다.
- String[] : 매개변수의 타입이 String[] 타입이다.
- args : args라는 이름의 매개변수이다.

main 메서드는 자바로 작성한 소스 코드 파일을 실행하면 가장 먼저 실행되는 메서드이다. 따라서 자바로 만들어진 어플리케이션이나 프로그램을 실행하면 main 메서드의 코드부터 호출되어 기능을 동작하는 것이다.
