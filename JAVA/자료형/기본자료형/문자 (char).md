char의 기본값 = '\u0000'

문자(정수)를 저장하는 자료형
문자를 저장하기 위해서는 작은따옴표 (' ')사용
유니코드 값을 그대로 입력 가능
정수값(10진수 또는 16진수 등 다양한 진법)을 직접 입력 가능

ex) 
	char a = 'A'
	char b = '\u0042'
	char c = '1';
	char d = 97;
	char e = 0xac00;
