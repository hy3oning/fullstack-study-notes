
String 클래스
- 문자열 저장 클래스 타입
- 문자열을 쌍따옴표""안에 표기

String 객체를 생성하는 2가지 방법

	String 참조변수명 = new String("문자열");
	String 참조변수명 = "문자열";
- new 키워드를 사용한 객체 생성
- 참조변수에 바로 문자열 리터럴 입력


String 클래스만 가지고 있는 2가지 특징
- 객체내의 값 변경 불가능 //값 변경시 새로운 객체를 생성하여 작성
- 리터럴을 바로 입력한 데이터는 문자열이 같은 경우 하나의 객체를 공유

ex)

	String str1 = new String("안녕"); //str1 스택영역 "힙영역"100번지
	String str2 = ("안녕"); //200번지
	String str3 = ("안녕"); //200번지
	String str4 = new String("안녕"); //300번지
