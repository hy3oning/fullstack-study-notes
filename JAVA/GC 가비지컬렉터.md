
==사용하지 않는 객체의 메모리를 GC가 주기적으로 검사해서 청소해준다.

>[!note]
>WHY?
>C와 C++같은 Unmanaged language는 free()와 같은 함수를 사용해서 메모리를 직접 메모리해제해야한다. 
>이런 번거로운 일을 GC가 대신 해주고 있는 것이다.
>사용하지 않는 객체의 메모리 점유는 결국 메모리 누수로 이어지게 된다.
>메모리는 한정된 자원이기 때문에 사용하지 않고나 필요가 없는 부분은 해제를 해주는 것이 맞다.


주요 기능
=
- 자동 메모리 관리 : 프로그램이 사용하지 않는 객체(가비지)를 자동으로 식별하고 메모리에서 제거합니다.
- 메모리 회수 : 사용하지 않는 메모리 공간을 회수하여 새로운 객체 할당에 사용할 수 있도록 합니다.
- 메모리 누수 방지 : 개발자가 실수로 메모리를 해제하지 않아 발생하는 메모리 누수 문제를 방지하는 데 도움을 줍니다.
