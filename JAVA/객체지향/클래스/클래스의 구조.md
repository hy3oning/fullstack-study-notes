- 필드 ==(heap메모리에 저장)==
	- 필드란 클래스에 포함된 변수를 의미한다. 객체의 상태(state)속성을 나타내며, ==멤버변수== 라고도 한다.
	1. [[인스턴스 변수]]
		- 인스턴스가 갖는 변수, 인스턴스가 생성될 때 생성된다. 서로 독립적인 값을 갖고있어 heap영역에 할당되고 'GC(가비지컬렉터)'에 의해 관리된다.
	 2. [[클래스 변수(static 변수)]]
		- ==static== 키워드가 인스턴스 변수 앞에 붙여진 변수를 말한다. 해당 클래스에서 파생된 인스턴스는 클래스 변수를 공유한다. 인스턴스가 생성되지 않아도 사용이 가능하다. ==this== 참조변수의 사용은 불가능하다. heap 영역이 아닌 static 영역에 할당되며 GC에 관리받지 않는다. public 키워드까지 붙인다면 전역변수라고 볼수도 있다.
	3. [[지역 변수]]
		- 특정[[메소드]] 내에서만 사용되는 변수 {중괄호 안}
	4. [[매개 변수 (prarmeter)]]
- 생성자
	- 인스턴스를 생성해주고 클래스의 객체를 초기화하는데 사용되는 특수한 메서드라고 할 수있다. 메서드와 달리 반환 값이 없기 때문에 return도 사용하지 않는다. ==클래스와 이름이 같아야 하며== 최소 1개 이상의 생성자를 가질 수 있다. 생성자가 한개도 정의되지 않은 경우 ==default==생성자를 자동으로 추가한다.
- 메소드(method)
	- 해당 객체의 행위를 나타낸다. 자바에서는 모든 메소드가 클래스에 존재하기 때문에 모든 함수가 메소드이다.
	1. [[인스턴스 메소드]]
	2. [[클래스 메소드 (Static)]]
	3. Getter
	4. Setter
	5. [[Overloading]]
	6. [[Overriding]]

1) 클래스는 접근제어자를 public 으로 한다.
2) 생성자는 접근제어자를 public 으로 한다. 
3) 멤버변수는 접근제어자를 private으로 한다.(캡슐화)
4) 멤버함수는 접근제어자를 public 으로 한다.
5) 클래스멤버변수와 클래스멤버함수의(static) 클래스명.클래스멤버변수
6) 인스턴스멤버변수와 인스턴스멤버함수는 객체참조변수명.인스턴스멤버변수
7) 클래스멤버함수는인스턴스멤버변수와 인스턴스멤버함수를 불러서 사용할 수 없다.
	(메모리에 없을 수 있기 때문 (객체를 언제생성할지 몰라서))
8) 인스턴스멤버함수는 모든 인스턴스변수,다른인스턴스멤버함수,클래스멤버변수,
    클래스멤버함수를 사용할수있다.
